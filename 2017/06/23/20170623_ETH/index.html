乙太幣 - 簡介

6月 23, 2017 分類 乙太幣

以太坊(Ethereum)，是一個開源的智慧合約的公共區塊鏈。

乙太幣(ETH)是以太坊(Ethereum)的數位貨幣，開發者們需要支付乙太幣(ETH)來進行應用。

通過其專用加密貨幣乙太幣（Ether）提供去中心化的虛擬機器(Ethereum Virtual Machin)來處理對等合約。

截至2017年6月，乙太幣是市值第二高的加密貨幣，僅次於比特幣。

簡介

ETH是一種點對點的電子現金系統

目前市面上絕大多數電子支付系統，都需要一個第三方(EX:銀行)來控管你的帳戶，所以你的錢其實都在第三方的掌握之中，

ETH目的在於創造一個無特定人可以控管的帳本系統，透過P2P網路交換電子簽章，並且會被收納到帳本裡。

透過一個『挖礦』的機制獎勵使用者驗證交易提高帳本安全性。

乙太幣的價值並不是因為這個記帳單位有多厲害多有擔保，發行銀行多強大，而是前所未有的『分散式機制』。

而不像現有的電匯那樣緩慢，或是Paypal的高成本，或是其他第三方支付有國家侷限性。

主要特色

  • 全球可及、快速、安全、隱私、低成本、所有人都可使用。
  • 智慧合約（smart contract）：儲存在區塊鏈上的程式，由各節點執行，需要執行程式的人支付手續費給結點的礦工或權益人。

乙太幣最重要的技術就是智慧合約。智慧合約是儲存在區塊鍵上的程式，可以協助和驗證合約的談判和執行。

挖礦 Mining

乙太幣這個記帳單位的產生來自於挖礦，而產出總量上限是固定的，約9000萬顆至1億顆。

礦工的工作和其實和銀行類似，會去驗證每一筆的交易的有效性，並且會去消耗運算資源產生帳本，使的帳本(區塊)難以任意偽造。

而礦工成功產生一本帳本後，也可以得到獎勵，因此大家就會搶著當礦工，並且讓產生帳本的難度提高，使得整個體系的安全度更高。

整個系統裡的帳本(區塊)不斷地產生，我們稱這個帳本系統叫做區塊鍊。

區塊鍊 Blockchain

礦工會藉由P2P網路不斷地接收到來自世界各地的交易，而符合規則的交易就會被納入到區塊裡，最基本的規則就是電子簽章要符合，就跟一般支票一樣，需要確認簽名是否正確。

礦工會不斷去挖礦去算工作量證明，當達到系統設定的標準，帳本難度夠高了。

那這本帳本就會被廣播出去，讓世界上所有的完整節點接收到新帳本。

假如你的交易有在這新的帳本(區塊)裡，那就是你的交易得到了確認，因為有效帳本只有一本，所以交易確認後就無法複製或偽造，

交易 Transaction

系統裡的交易可以把它想像成就是在簽支票，當別人給我們了有效的乙太幣支票。

那我們的帳戶裡就會顯示有乙太幣，這個時候我們就可以簽支票給別人。

這個時候就會需要用到我們的印章(私鑰)進行簽章，確保我們簽出去的支票是有效的。

同樣我們也會把這筆交易廣播出去。

礦工會用公鑰確認這張支票是不是真的是對應私鑰所簽發，檢查付的手續費夠不夠。

所有標準都有達到就會加到自己的帳本中。當新區塊產生時，你的交易就可以得到確認。
